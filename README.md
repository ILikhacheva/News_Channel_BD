
# News Channel BD

Минимальный новостной сайт с категориями, авторизацией и добавлением новостей.
Minimal news site with categories, authorization, and news adding.



## Запуск / Launch

1. Установите зависимости:  
   Install dependencies:
   ```bash
   npm install
   ```
2. Запустите сервер:  
   Start the server:
   ```bash
   node server.js
   ```
3. Откройте `index.html` в браузере.  
   Open `index.html` in your browser.
